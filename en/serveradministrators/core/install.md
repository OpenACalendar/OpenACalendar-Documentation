# Install

## Requirements


  *  PHP 5.4+
  *  Postgresql
  *  Apache with ModRewrite (or equivalent)
  *  Command line access
  *  Cron

(It has currently been tested on Linux webservers only, tho it may be possible 
to get it working on Windows servers.)
